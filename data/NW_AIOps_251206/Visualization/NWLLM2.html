기존 AI/ML (Traditional)

태스크 종속성 (Task-Specific):
트래픽 예측, 침입 탐지 등 특정 목적마다 별도 모델 개발 필요

높은 재학습 비용:
네트워크 환경 변화 시 모델 재학습 필수 (Lack of Generalization)

비정형 데이터 한계:
로그, 매뉴얼 등 텍스트 기반 정보 해석의 어려움

LLM (Next-Gen)

범용 추론 능력 (Zero-shot):
방대한 도메인 지식 기반, 추가 학습 없이 즉각적 상황 판단

자연어 → 코드 변환:
운영자의 의도(Intent)를 이해하고 실행 가능한 설정 코드로 자동 생성

통합 운영 관리:
분석, 구성, 최적화를 단일 모델로 처리하여 복잡성 해결
